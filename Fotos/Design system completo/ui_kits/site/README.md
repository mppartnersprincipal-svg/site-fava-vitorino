# UI Kit — Site institucional

Site institucional do escritório Fava e Vitorino Advocacia (novo design, autorado a partir das fundações da marca — não é recreação de site existente).

- `index.html` — protótipo interativo: navegação Início / Áreas / Contato funcional, formulário e diálogo de agendamento fake.
- `SiteApp.jsx` — roteador simples entre telas.
- `SiteHeader.jsx` / `SiteFooter.jsx` — moldura do site.
- `SiteHome.jsx` — hero verde, áreas em destaque, como atuamos, CTA.
- `SiteAreas.jsx` — as 10 áreas de atuação.
- `SiteContato.jsx` — formulário de contato.

Fotos: blocos de placeholder marcados "FOTO" — substituir por fotografia arquitetônica sóbria (ver readme raiz).
